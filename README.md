### 使用說明

有 `編輯` 及 `檢視` 兩種模式
* 編輯：
  * 加入投籃位置
  * 修改球員資料
  * 修改分佈圖名稱
  * 新增或刪除分佈圖
* 檢視：
  * 檢視編輯完的分佈圖
  * 可以合併分佈圖
